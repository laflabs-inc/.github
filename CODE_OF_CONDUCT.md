# Code of Conduct

LafLabs는 건강하고 전문적인 기술 커뮤니티를 지향합니다.

## Our standards

다음을 기대합니다.

- 사람과 아이디어를 구분하여 토론합니다.
- 기술적 비판은 구체적인 근거와 대안을 중심으로 합니다.
- 차별, 괴롭힘, 위협, 모욕적인 개인 공격을 허용하지 않습니다.
- 개인정보와 보안 정보를 존중합니다.
- 실수와 질문을 조롱하지 않습니다.
- 공개 채널에서 해결하기 어려운 갈등은 Maintainer에게 에스컬레이션합니다.

## Unacceptable behavior

다음 행동은 허용되지 않습니다.

- 지속적인 괴롭힘 또는 위협
- 혐오 표현과 차별
- 원치 않는 개인정보 공개
- 악의적인 보안 정보 공개
- 서비스 또는 커뮤니티 운영을 고의로 방해하는 행위
- 반복적인 스팸과 악의적인 선동

## Enforcement

Maintainer는 상황에 따라 다음 조치를 취할 수 있습니다.

- 안내 또는 경고
- Content 수정 또는 삭제
- 일시적 참여 제한
- 영구적 참여 제한
- GitHub 또는 관련 Platform에 신고

집행 과정에서는 사건의 심각도, 반복성, 사용자와 프로젝트에 대한 위험을 고려합니다.

## Scope

이 Code of Conduct는 LafLabs가 관리하는 공개 Repository, Issue, Pull Request, Discussion 및 관련 공식 커뮤니티 공간에 적용됩니다.
